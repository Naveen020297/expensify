# Expense Tracking Feature

## Overview
This document outlines the implementation of the expense tracking feature in the application.

## Features
- Add, view, edit, and delete expenses.
- Categorize expenses.

## Setup
1. Install dependencies: `npm install`
2. Run the application: `npm start`

## CI/CD
This project uses GitHub Actions for CI/CD. The pipeline is defined in `.github/workflows/ci-cd.yml`.
